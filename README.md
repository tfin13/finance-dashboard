A full-stack web app built with **Flask, SQLAlchemy, Bootstrap, and Matplotlib** to track personal finances.

## Features
- Add transactions with amount, category, and description
- View transactions in a clean Bootstrap table
- Auto-generated pie chart of spending by category
- Dark mode toggle for modern UX
- Summary section showing total spending

## Tech Stack
- Python (Flask, SQLAlchemy)
- HTML, CSS, Bootstrap
- Matplotlib (charts)
- SQLite (database)
